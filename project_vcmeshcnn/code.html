Code (unofficial version)

Since the official version of code on Facebook github doens't provide running instructions yet, we provide an unofficial version here. Compared with the official version, the unofficial one has full instructions for setting up environments and running. It also contains the code for comparisons with other Graph CNNs. One might still want to use the official one for large meshes because it is faster and more memory efficient and support training with multiple GPUs.
